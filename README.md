# PyDrill — Python Practice Tool

A simple browser-based tool to practice Python concepts interactively.

## Why this was created

I am preparing for a **Product Analyst L2 interview** at a healthcare analytics company. The role requires hands-on knowledge of Python and SQL for data pipelines, reporting automation, and analysis.

Instead of just reading theory, I wanted a tool where I can:

- Practice Python functions by actually typing answers
- Test myself on strings, lists, sets, dictionaries, and built-in functions
- Track how many questions I get right and wrong
- Jump to any specific question I want to revisit
- Use it anytime from a browser without installing anything

## What it covers

- String functions — `strip`, `split`, `join`, `replace`, `find`, `upper`, `lower` and more
- List functions — `append`, `pop`, `sort`, `slice`, `comprehension` and more
- Set operations — `union`, `intersection`, `difference`, `discard` and more
- Dictionary functions — `get`, `keys`, `values`, `items`, `pop`, `update` and more
- Built-in functions — `map`, `filter`, `zip`, `enumerate`, `any`, `all` and more

## How to use

Just open `index.html` in any browser. No installation needed.

**Three practice modes:**
- ⌨️ **Type Answer** — type the output and check if you are correct
- 💡 **Hint → Reveal** — see a hint first, then reveal the full answer
- 🃏 **Flashcard** — flip the card to see the answer

## Plan

This tool will grow as I learn more topics. SQL practice questions will be added next.
